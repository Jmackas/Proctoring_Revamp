Frequently asked questions for the Proctoring Software


I accidentally disallowed Chrome from accessing my Microphone and Camera, how do I reenable them?
  1. Select the customise dots on the top right of the Chrome browser.
  2. Select 'Settings' in the drop down menu.
  3. Scroll down the page that opens, and select 'Advanced'
  4. Select 'Content settings' under 'Privacy and security'
  5. Select 'Camera' and 'Microphone' to determine if the Chrome extension has been blocked.
  6. Select the trashcan icon next to 'CRIS'.

When the proctoring page opens on the test again, it will now ask if you wish to allow access. Remember to select 'Allow'.
